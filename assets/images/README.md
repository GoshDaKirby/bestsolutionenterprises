# Background Images

Place the following two files in this folder:

| File      | Used on          | Notes                                      |
|-----------|------------------|--------------------------------------------|
| `bg.gif`  | Home page only   | Animated — fills the full 100vh hero       |
| `bg.png`  | All other pages  | Static freeze-frame of the same image      |

Both are referenced from CSS as:
  `url('../images/bg.gif')`
  `url('../images/bg.png')`

Recommended: high-resolution (1920×1080+), landscape orientation.
The dark navy overlay will be applied automatically via CSS — no editing needed.
